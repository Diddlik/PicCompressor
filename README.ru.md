# PicCompressor

Локальное и конфиденциальное сжатие JPEG для настольного приложения и командной строки.

## О PicCompressor

PicCompressor преобразует изображения JPEG и PNG в компактные JPEG-файлы с помощью Jpegli. Обработка выполняется на вашем компьютере; изображения никуда не загружаются.

Настольное приложение и CLI используют общий процесс сжатия и одинаковые правила безопасности.

## Возможности

- Сжатие JPEG с помощью Jpegli
- Настольный интерфейс и CLI для автоматизации
- Отдельные файлы, пакетная обработка и рекурсивный поиск в папках
- Перетаскивание файлов, отмена, повтор и локальная история
- Настраиваемое качество, субдискретизация цветности и прогрессивное кодирование
- Политики конфиденциальности EXIF и цветовых профилей
- Проверка временного файла перед публикацией результата
- Защита от конфликтов и явное управление перезаписью
- Удобочитаемый и версионированный JSON-вывод CLI

## Текущее состояние

PicCompressor находится на стадии альфа-разработки. В настоящее время проверена Windows x64. Поддержка macOS, Linux и других архитектур запланирована, но ещё не подтверждена.

Интерфейс приложения сейчас поддерживает английский и немецкий языки. Перевод README на русский язык не означает, что интерфейс уже локализован на русский.

Публичные установщики пока не предлагаются: необходимо завершить цифровую подпись релизов и оставшиеся проверки безопасности.

## Быстрый старт

Требования для текущей сборки под Windows:

- .NET SDK `10.0.302`
- Visual Studio с компонентами разработки классических приложений .NET и C++
- CMake `3.25` или новее
- PowerShell 7

```powershell
dotnet build PicCompressor.slnx
dotnet run --project src/PicCompressor.Desktop/PicCompressor.Desktop.csproj
```

При первой сборке загружаются и компилируются закреплённые исходники Jpegli.

## CLI

```text
piccompressor <входной_путь> [<входной_путь> ...] [параметры]
```

Пример:

```powershell
piccompressor "C:\Images" --recursive --quality 80 --output-dir "C:\Images\compressed"
```

Показать все параметры:

```powershell
piccompressor --help
```
